# Introdução à Ciência da Computação - Lab 8

## Descrilção da atividade

Neste laboratório, os alunos irão utilizar uma LLM de sua escolha para resolver pequenas tarefas acadêmicas e computacionais. O objetivo não é apenas obter respostas, mas aprender a interagir criticamente com a ferramenta.

Ao final da atividade, espera-se que o aluno saiba:
- utilizar uma LLM de forma consciente;
- escrever prompts mais claros e objetivos;
- comparar diferentes versões de prompts;
- avaliar criticamente as respostas geradas;
- identificar limitações, erros ou alucinações;
- relatar corretamente o uso da LLM no trabalho.

## Formato da entrega

 *relatorio.pdf*: Contém todos os exercícios, aprensentado prompt utilizado, resposta da LLM e reflexões
 *prompts.txt*: Documento com todos os prompts utilizados na tarefa.
